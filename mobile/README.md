# Coder Chat Mobile (Expo)

This folder contains the React Native + Expo mobile version of the admin dashboard.

## Features included
- Admin login flow (simulated auth)
- Add user to admin
- Sales list
- User registrations chart bars
- KPI dashboard for:
  - Monthly Recurring Revenue (MRR)
  - Number of customers
  - Churn rate
  - Trial-to-paid conversion rate
  - Customer acquisition cost (CAC)
  - Lifetime value (LTV)
  - Growth rate

## Run locally
```bash
cd mobile
npm install
npm run start
```

Then press:
- `a` for Android emulator
- `i` for iOS simulator (macOS)
- `w` for web
